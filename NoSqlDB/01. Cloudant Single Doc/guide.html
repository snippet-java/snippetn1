Cloudant NoSQL DB provides access to a fully managed NoSQL JSON data layer that's always on. This service is compatible with CouchDB, and accessible through a simple to use HTTP interface for mobile and web application models

Username and password need to be obtained by creating Cloudant NoSQL DB Service at :Cloudant NoSQL DB Service

  1. When you run the code the username and password is expected to be set for the Cloudant NoSQL DB Service to be used.
  2. Go to cloudant dashboard (http://username.cloudant.com/dashboard.html) and create a database with a few documents
  3. Replace Doc ID with _ID field of Cloudant document and execute. It retrieves all fields with values

For documentation visit IBM Cloudant Documentation
