1.1 Variables and Strings
A variable is a name that can reference a specific value. Variables are declared using var followed by the variable's name.

Here's an example:
var example;

The above variable is declared, but it isn't defined (it does not yet reference a specific value).

Here's an example of defining a variable, making it reference a specific value:
var example = 'some string';

A string is a sequence of characters.Examples of characters are letters, numbers, punctuation marks, and spaces.String values are surrounded by either single or double quotation marks. 
'this is a string' 
"this is also a string"
