Conversion between Strings and Numbers
Sometimes you will need to turn a number into a string.In those instances you will use the toString() method. Here's an example: 
            var i = 256;
            i = i.toString();

Similary the strings can also be converted into numbers.The Number() function converts the object parameter to a number that represents the object's value. If the value cannot be converted to a legal number, NaN is returned. 
            var str = "256";
            var i = Number(str);
